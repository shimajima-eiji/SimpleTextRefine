執筆系作業の添削補助ツール。
テキスト選択範囲をGPT APIに流して、応答を別ファイルに書き出し、diff viewerで差分表示する。
同階層または上位ディレクトリに.promptというファイルを事前に作ると、それがsystem promptとして渡される。.promptには複数のプロンプトを記録でき、呼び出し時に選択できる。

```yml
- |
  1つ目のプロンプト
- |
  2つ目のプロンプト
  複数行でも良い
```

# 開発時
## テスト実行
- .vscode/launch.jsonの効果で、F5 を押したときに起動する

## ビルド → 配布
- `npx vsce package`
